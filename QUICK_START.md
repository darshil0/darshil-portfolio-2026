# ⚡ Quick Start: GitHub Pages Deployment (5 Minutes)

## 🎯 Super Simple Method (Recommended for Beginners)

### **Step 1: Create a GitHub Account** (2 minutes)
1. Go to https://github.com
2. Click "Sign up"
3. Follow the signup process
4. Verify your email

---

### **Step 2: Create Your Repository** (1 minute)
1. Sign into GitHub
2. Click **+** icon (top right) → **"New repository"**
3. Fill in:
   - **Repository name**: `darshil-portfolio` (or any name)
   - **Description**: "Professional QA Engineering Portfolio"
   - **Public**: ✅ Select this
   - **Do NOT initialize** with README or .gitignore

4. Click **"Create repository"**

---

### **Step 3: Upload Your Files** (1 minute)
1. You'll see a screen that says "...or upload an existing file"
2. Click **"uploading an existing file"** link
3. Drag and drop your files:
   - `index.html` ⭐ IMPORTANT
   - `Darshil_Shah_QA_Engineering_Portfolio.pdf`
   - `README.md`
   - `portfolio-website.jsx`
   - `.gitignore`

4. Click **"Commit changes"**

---

### **Step 4: Enable GitHub Pages** (1 minute)
1. Go to repository settings: Click **Settings** (top right)
2. Click **"Pages"** on the left sidebar
3. Under "Build and deployment":
   - **Source**: Select **"Deploy from a branch"**
   - **Branch**: Select **"main"** and **"/ (root)"**
   - Click **"Save"**

4. Wait 2-5 minutes (GitHub builds your site)

---

### **Step 5: Access Your Live Website** (Instant!)
GitHub will show you the URL:
```
https://YOUR-USERNAME.github.io/darshil-portfolio/
```

**That's it! Your portfolio is live! 🎉**

## ❓ Troubleshooting

### "Page not found" error?
- ✅ Wait 5 minutes (GitHub Pages needs time to build)
- ✅ Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- ✅ Check URL is exactly right
- ✅ Make sure repository is PUBLIC

### PDF not downloading?
- ✅ Check PDF file is uploaded to GitHub
- ✅ Check filename is spelled correctly
- ✅ Try direct PDF link in browser

### Changes not showing?
- ✅ Wait 5 minutes
- ✅ Hard refresh page
- ✅ Clear browser cache
- ✅ Try a different browser
